# EPL Student Starter

Welcome to the **Practical OpenWrt Tradecraft** engagement-platform labs.

This repo contains everything you need to run the lab environment:

- **`.devcontainer/`** — VS Code / GitHub Codespaces devcontainer configuration.
- **`labs/`** — Lab exercises, supporting files, and the Mango overlay.

## Getting started

The fastest path is **GitHub Codespaces** (no local Docker required):

1. Click the **Open in Codespaces** button on
   [course.eplabs.cloud](https://course.eplabs.cloud) next to the lab you are
   working on.
2. Wait ~5 minutes for the Codespace to build (subsequent opens are instant).
3. Follow the lab instructions in the browser.

For local Docker, see the onboarding guide at
[course.eplabs.cloud/onboarding](https://course.eplabs.cloud/onboarding).

## Free-tier note

GitHub Codespaces personal free tier gives you **120 core-hours/month** — more
than enough for a 2-day workshop (~28 core-hours on a 2-core machine). If you
are a student, check [GitHub Education](https://education.github.com) for
additional allowances.

## Synced from private master

This repository is automatically synced from the instructor's private master
on every push. Pull to get the latest lab content:

```sh
git pull
```
